
# Neo4j GraphQL Roadtrip

A migration tool for schema definition for `@neo4j/graphql`.

Usage:

`roadtrip typedefs.graphql -o new_typdefs.graphql`

Options:

-   -o, --output <string>
-   --subscriptions enable subscriptions
-   -h, --help display help for command
